# Zolt

Zolt is a quiz software for schools, colleges, and other organizations. It helps run quiz competitions that are automated, transparent, and customizable.

## Features

*   **Customizable Rounds:** Adjust points, passing criteria, and use confirmation alerts to prevent mistakes.
*   **12 Themes:** Includes 6 light and 6 dark themes to change the look of the app.
*   **Adjustable Layout:** The top bar, side bar, and bottom bar can be moved or hidden.
*   **Local Storage:** All data is saved locally using a SQLite database. No internet connection is needed to use the app.
*   **Score Transparency:** A score log records every point change, making it easy to review and resolve any disputes.
*   **Modern Design:** The app is designed to be easy to use and visually clear.

## Tech Stack

*   **Backend & Desktop:** Electron, TypeScript
*   **Frontend:** React 19, Tailwind CSS, shadcn/ui
*   **Database:** SQLite
*   **Key Libraries:** Framer Motion, dnd-kit, React Dropzone, Zod

## Getting Started

To run the application on your computer, follow these steps:

1.  Make sure you have Node.js and npm installed.
2.  Open two separate terminal windows.
3.  In the first terminal, run the frontend:
    ```bash
    cd frontend
    npm install
    npm run dev
    ```
4.  In the second terminal, run the main application:
    ```bash
    npm install
    npm run dev
    ```
