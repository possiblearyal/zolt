# Zolt Frontend

This directory contains the source code for the Zolt user interface.

## Description

The frontend is built with React and provides all the visual components and user interactions for the Zolt application. It is responsible for the quiz experience, from setup to final scores.

## Key Technologies

*   React 19
*   TypeScript
*   Tailwind CSS
*   shadcn/ui
*   Framer Motion
*   dnd-kit
*   React Dropzone
*   Sonner (for notifications)
*   Zod (for data validation)
